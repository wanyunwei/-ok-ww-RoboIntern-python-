# -ok-ww-RoboIntern-python-
本代码的作用是自动打鸣潮每日

本人学生，写得很烂，只是记录一下

逻辑链：用RoboIntern设置项目自动启动时间（电脑需开机） -- 项目启动，自动打开并自己打每日  --  项目关闭，自动关闭鸣潮

如果你想用我的代码的话

1.你需要在源码中自己配置鸣潮游戏路径，以及okww的路径，并且自己打包，放wuwaAuto图片文件夹旁边

（

打包过程：

先导入库pip install pyinstaller

然后输入pyinstaller -F auto2.0.py

）

2.你需要把ok-ww调成暗色

3.你需要把鸣潮游戏内分辨率改成1920×1080

4.你的大世界队伍必须有强度打每日，一旦角色死亡，将会失效（所以建议带守岸人）

5.电脑需要开机

6.暂不支持版本更新或者热更新情况下的每日（后者打算遇到了更新，前者不打算更新了）
